# Forecasting Financial Inclusion

This project analyzes financial inclusion data for Ethiopia using unified datasets.

## Structure
- `data/`: Contains the raw data files.
- `src/`: Source code for data loading and enrichment.
- `notebooks/`: Jupyter notebooks for EDA.

## Setup
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Tasks
- **Task 1**: Data Exploration and Enrichment (Branch: `task-1`)
- **Task 2**: Exploratory Data Analysis (Branch: `task-2`)
